# sentiment_analysis_using_NLP

This project contains the notebook **`sentiment_analysis_using_NLP.ipynb`**, which demonstrates how to perform sentiment analysis using natural language processing techniques.

## Requirements

* Python 3
* NumPy
* Pandas
* Scikit-learn
* NLTK or spaCy
* Matplotlib

## Usage

1. Open the notebook:

   ```bash
   jupyter notebook sentiment_analysis_using_NLP.ipynb
   ```
2. Run all cells to preprocess text, train the model, and evaluate sentiment classification.

## Description

The notebook includes:

* Text cleaning and tokenization
* Feature extraction (e.g., TF-IDF)
* Training ML models for sentiment classification
* Evaluation and visualization
